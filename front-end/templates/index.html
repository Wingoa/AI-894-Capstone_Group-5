About UFC Fighter Optimizer



Style-aware matchup analysis, outcome prediction, and coaching insights.

We ingest per-fight statistics from public event pages and persist them as append-only CSV caches for reproducibility. From this data we derive interpretable fighter "style" vectors, train outcome models, and produce per-prediction explanations to highlight key matchup drivers. The UI focuses on fast comparatives, visual explanations, and simple simulations you can run in the browser.

This project is intended as an MVP research tool: reproducible scraping, compact CSV caches, and interpretable features so coaches and analysts can validate model signals against video and domain knowledge.

Data

Append-only CSV caches store scraped fight stats (strikes, takedowns, control time, pace), with timestamps for reproducibility and time-split evaluation.



Style Modeling

NMF decomposes per-fight metrics into concise, interpretable style components used as model features and scouting signals.



Outcome Prediction

Predictive models are trained on time-held-out splits; SHAP explanations surface the top drivers behind each prediction.



Predictions are probabilistic guidance, not guarantees. Use them alongside video review and domain expertise.
