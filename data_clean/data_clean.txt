DATA CLEANING MODULE
====================

Transforms raw UFC fight stats into ML-ready features.

FILES
-----
explore_data.py  - View raw data
process_data.py  - Parse, normalize, create train/test split
features.py      - Feature selection (NMF & XGBoost)

USAGE
-----
python process_data.py

Output: 5 CSVs in src/main/resources/clean_data/
  - train_set.csv (pre-2025 data)
  - test_set.csv (2025+ data)
  - Plus 3 intermediate files

FEATURES
--------
XGBoost: 14 features (sig_str_per_min, td_per_min, ctrl_pct, etc.)
NMF: 25 features (raw striking/grappling stats)
